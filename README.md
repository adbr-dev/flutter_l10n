# l10n

flutter 국제화 프로젝트

## Getting Started

0. new project 
    ```shell
    flutter create l10n
    ```

1. add flutter package 
  * [intl 0.16.1](https://pub.dev/packages/intl/install) 
  * [intl_translation: ^0.17.10+1](https://pub.dev/packages/intl_translation/install)
  ```yaml
  dependencies:
    flutter:
      sdk: flutter
    flutter_localizations:
      sdk: flutter
    intl: ^0.16.1

  dev_dependencies:
    flutter_test:
      sdk: flutter
    intl_translation: ^0.17.10+1
  ```
